# 🎯 Number Guessing Game (C++)

## 📌 Project Description

The **Number Guessing Game** is a simple console-based game developed in C++.
The program generates a random number within a selected range, and the player must guess the correct number within limited attempts.

This project demonstrates the use of:

* Loops
* Conditional statements
* Switch case
* Random number generation
* Basic game logic

---

## 🚀 Features

* 🎮 Three Difficulty Levels:

  * Easy (1 – 50)
  * Medium (1 – 100)
  * Hard (1 – 500)

* 🔢 Random number generation using `rand()`

* ⏳ 7 attempts per game

* 📢 Hints after every guess:

  * "Too High"
  * "Too Low"

* 🏆 Win message on correct guess

* 💀 Game Over message if attempts are finished

* 🎨 ASCII styled game title

* 🔁 Continuous play until user exits

---

## 🎮 How to Play

1. Run the program.
2. Select difficulty level.
3. Enter your guesses.
4. You have 7 attempts.
5. The game will guide you:

   * If guess is smaller → "Too Low"
   * If guess is bigger → "Too High"
6. Win by guessing correctly.
7. Lose if attempts finish.
